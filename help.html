Main Access Modifiers

Col.fSelect( int index )

takes an int as index and returns the column at position index from the data model

Col.fString( String str )

takes a String literal an returns it in the output table

Col.fNumber( Number aNumber )

takes a Number as argument and returns it in the output table

Col.fBoolean( Boolean bol )

takes a Boolean value (true/false) as argument and returns it in the output table

String Functions

Col.fUpCase( String str )

Converts a string to uppercase letters

Col.fLowerCase( String str )

Converts a string to lowercase letters

Col.fIndexOf( String str , String str2 )

Returns the position of the first found occurrence of a specified value in a string

var col1 = new Col.fString("lmn");
var col2 = new Col.fString("abcdefghijklmnop");
var sum = new Col.fIndexOf(col2,col1);
 returns 11

Arithmetic Functions

Col.fAddition( int i , int j )

returns i + j

Col.fSubtraction( int i , int j )

returns i - j

Col.fMultiplication( int i , int j )

returns i * j

Col.fDivision( int i , int j )

returns i / j

Col.fModulus( int i )

returns modulus of i

Col.fToExponential( int i )

returns Exponential of i

Comparison Functions

Col.fEquals( Obj b1 , Obj b2 )

Returns b1 == b2. Compares two objects and returns true if they are equal

Col.fNEquals( Obj b1 , Obj b2 )

Returns b1 != b2. Compares two objects and returns true if they are not equal

Col.fAnd( Boolean b1, Boolean b2 )

logical and operator. T and T = T . T and F = F. ect.

Col.fOr( Boolean b1, Boolean b2 )

logical or operator. T or T = T . T or F = T. ect.

Col.fGt( Obj o1 , Obj o2 )

compares if o1 is grether ( > ) than o2

Col.fLt( Obj o1 , Obj o2 )

compares if o1 is less ( < ) than o2
