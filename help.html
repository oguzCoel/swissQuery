Main Access Modifiers

Col.fSelect( int index )

takes an int as index and returns the column at position index from the data model

Col.fString( String str )

takes a String literal an returns it in the output table

Col.fNumber( Number aNumber )

takes a Number as argument and returns it in the output table

Col.fBoolean( Boolean bol )

takes a Boolean value (true/false) as argument and returns it in the output table
